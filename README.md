# angola-banking-customer-feedback-insights
Prompt de Inteligência Artificial para análise de feedbacks de clientes bancários no mercado angolano. Identifica temas, sentimento, urgência, padrões e oportunidades de melhoria em canais digitais, Multicaixa Express, cartões, ATM, TPA, transferências e atendimento, transformando dados não estruturados em insights acionáveis.
